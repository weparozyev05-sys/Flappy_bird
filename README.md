# Unity Flappy Bird

A 2D Flappy Bird clone developed with Unity and C#.

## Features
- Player movement and jump mechanics
- Rigidbody2D physics
- Collision detection
- Score system
- High score saving using PlayerPrefs
- Sound effects
- Dynamic difficulty increase
- UI (Play button, Game Over screen)

## Technologies Used
- Unity 6
- C#
- TextMesh Pro
